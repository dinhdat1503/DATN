## Hướng dẫn khắc phục lỗi "File not found" của Wikipedia:
1. **Sử dụng link ở cột "Trang hiển thị ảnh chính thức":** Các link này dẫn đến trang web mô tả của Wikimedia Commons (chứ không phải link tải ảnh thô dễ bị lỗi server). Trang web này luôn hiển thị ảnh ở trung tâm, bạn chỉ cần click chuột phải vào ảnh trên trang web đó và chọn **"Save image as..."** (Lưu hình ảnh thành...) để tải về.
2. **Sử dụng link ở cột "Link tìm kiếm nhanh trên Google Images":** Click vào link này $\rightarrow$ Trình duyệt tự tìm kiếm hình ảnh đó trên Google. Bạn có thể chọn bất kỳ hình ảnh nào cùng nội dung có chất lượng cao nhất để tải về.
3. **Chụp màn hình từ PDF gốc:** Nhấp vào cột **"Link xem bài báo gốc"** để mở bài báo $\rightarrow$ Dùng công cụ **Snipping Tool** của Windows chụp lại hình tương ứng.
